Population


In statistics, a population refers to the entire set of individuals, items, or events of interest for a particular study. It is the complete group from which data is collected or analyzed. The population can be finite (with a limited number of elements) or infinite (with theoretically unlimited members).

Statistical Units


A statistical unit is the basic entity or object on which data is collected in a statistical study. Each unit provides one set of data points for analysis. Statistical units are often referred to as individuals, observations, or data points.

Distribution


A distribution in statistics refers to how values of a variable or a set of variables are spread or arranged over possible values. It shows how often each value occurs (frequency) or how likely each value is (probability). The distribution gives insight into the shape, central tendency, and spread of the data.

Notion of Average

The average (or mean) is a fundamental concept in statistics, representing the central or typical value of a set of data. There are several types of averages, but the most common is the arithmetic mean.

Arithmetic Mean:

The sum of all the values divided by the number of values.

Formula:
Mean = (x1 + x2 + ⋯ + xn) / n

Where x1, x2, …, xn are the data points, and n is the number of data points.
